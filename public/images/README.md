# Gallery images

Add 4–6 images here to show in the app’s gallery. The app expects these filenames:

- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`
- `photo-5.jpg`
- `photo-6.jpg`

You can use 4, 5, or 6 of them; the gallery will show whichever files exist. Use `.jpg` or `.jpeg` (or update the paths in `src/components/Gallery.tsx` if you prefer `.png`).
